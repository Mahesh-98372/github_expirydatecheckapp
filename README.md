# Expiry Date Checking System

A Java-based application that helps users track product expiry dates and receive alerts before products expire.

## Features

- Add and manage products
- Track expiry dates
- Search products
- Expiry alerts
- User-friendly interface
- SQLite/MySQL database support

## Technologies Used

- Java
- Android Studio
- SQLite
- MySQL
- XML
- Git & GitHub

## Project Structure

```
Expiry-Date-Checking-System/
│── src/
│── database/
│── images/
│── README.md
```

## Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Expiry-Date-Checking-System.git
```

2. Open the project in Android Studio.

3. Build and Run the project.
